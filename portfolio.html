Recipe Box

This full stack blog application allows a user to search recipes that have been inputted into the application, create their own recipes to share, view favorite recipes, and share recipes by email or social media.

This application was build using:

  • HTML5
  • CSS
  • Bootstrap
  • jQuery
  • JavaScript
  • NodeJS
  • MySQL
  • Sequelize
  • Express
  • Express-fileupload
  • Nodemailer
